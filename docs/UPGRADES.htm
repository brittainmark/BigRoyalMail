How to upgrade to the latest Version of Big Royal Mail

IMPORTANT: Version 3.7.0 onwards has been designed to work with Zen Cart V1.5.8 upwards If you are using a version of zen cart before this it will not function correctly.
However you can still use the YYYYUpdate.sql files to update your rates.

Note: this upgrade will remove all handling fees and override changes to country codes you may have made. So please note any changes you have made before you begin.

  4. Copy the files from BigRoyalMail_V3.7.0/includes/* and BigRoyalMail_V3.7.0/images/* onto your server into:
    YOUR_SHOPPING_CART_DIRECTORY/includes/
    YOUR_SHOPPING_CART_DIRECTORY/images/
  5. Uninstall and reinstall all the Royal mail shipping modules you are using.
    1. Login to your zen cart admin
    2. Use Admin > Modules > Shipping
    3. For each of the modules installed.
      Green dot in sort order: installed and in use.
      Yellow dot in sort order: Installed not in use.
    4. Highlight by clicking the module line.
    5. Click Remove Module
    6. Click Remove Module on the confirmation message
    7. Click Install Module
      NB. If the dot after status was Yellow you do not need to reinstall as you were not using it so skip remaining steps for that module.
    8. Modify Enable this shipping option to True to turn the module on
    9. If you do not want to display invalid shipping options
      Modify Hide Invalid Shipping to True
    10. Check the Handling fee/s and modify if necessary
    11. Click Update
  6. If you use printed paper rates
    After You have installed all the modules you need.
    Run PrintedPapers.sql
    1. Backup your database before running update.
    2. Login to your zen cart admin
    3. Use Tools > Install SQL Patches
    4. Copy the contents of PrintedPapers.sql and paste into the text window
      or
      User the Browse button to load PrintedPapers.sql into zen cart.
    5. Press send button to execute script
