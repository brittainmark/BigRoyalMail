From Version 3.4.0 any

  1. Copy the files from BigRoyalMail_V3.4.1/includes/* and BigRoyalMail_V3.4.1/images/* onto your server into:
    YOUR_SHOPPING_CART_DIRECTORY/includes/
    YOUR_SHOPPING_CART_DIRECTORY/images/
  2. Run 2020Update.sql
    1. Backup yuor database before running update.
    2. If you use handling fees use the editor of your choice to set the correct handling fees. Save the changes
    3. Login to your zen cart admin
    4. Use Tools>Install SQL Patches
    5. Copy the contents of 2020Update.sql and paste into the text window
      or
      User the Browse button to load 2020Update.sql into zen cart.
    6. Press send button to execute script
  3. If you use printed paper rates
    Run PrintedPapers.sql
    1. Login to your zen cart admin
    2. Use Tools>Install SQL Patches
    3. Copy the contents of PrintedPapers.sql and paste into the text window
      or
      User the Browse button to load PrintedPapers.sql into zen cart.
    4. Press send button to execute script

    4. From Version 3.3.2 to 3.3.9

    1. Run RemoveUSA.sql
      This removes the two letter country code 'US' from all Royal mail modules (not parcel force)
      NB. International Signed for is no longer valid to USA.
      1. Login to your zen cart admin
      2. Use Tools>Install SQL Patches
      3. Copy the contents of RemoveUSA.sql and paste into the text window and click Send
        or
        User the Browse button to load RemoveUSA.sql into zen cart and click Upload.
      4. Press send button to execute script
    2. Copy the files from BigRoyalMail_V3.4.1/includes/* and BigRoyalMail_V3.4.1/images/* onto your server into:
      YOUR_SHOPPING_CART_DIRECTORY/includes/
      YOUR_SHOPPING_CART_DIRECTORY/images/
    3. Uninstall and reinstall all the Royal mail shipping modules you are using.
      1. Use Modules>Shipping
      2. For each of the modules installed.
        Green dot in sort order: installed and in use.
        Yellow dot in sort order: Installed not in use.
      3. Highlight by clicking the module line.
      4. Click Remove Module
      5. Click Delete on the confirmation message
      6. Click Install Module
        NB. If the dot after status was Yellow you do not need to reinstall as you were not using it so skip remaining steps for that module.
      7. Modify Enable this shipping option to True to turn the module on
      8. If you do not want to display invalid shipping options
        Modify Hide Invalid Shipping to True
      9. Check the Handling fee/s and modify if necessary
      10. Click Update
      11. Strictly speaking it is only necessary to perform these steps for modules that post to USA
        (rmamlargeletter, rmamletter, rmamparcel, rmamtlargeletter, rmamtletter, rmamtparcel, rmamtslargeletter, rmamtsletter, rmamtsparcel)
    4. If you use printed paper rates
      Run PrintedPapers.sql
      1. Login to your zen cart admin
      2. Use Tools>Install SQL Patches
      3. Copy the contents of PrintedPapers.sql and paste into the text window
        or
        User the Browse button to load PrintedPapers.sql into zen cart.
      4. Press send button to execute script
